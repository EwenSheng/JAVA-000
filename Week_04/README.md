1.（选做）列举常用的并发操作 API 和工具类，简单分析其使用场景和优缺点。

~~~
    CountDownLatch
    
    适用场景：在多线程执行过程中设置几个门闩，当所有的门闩被打开时，被挡在门外的线程才能继续执行。
    
    CyclicBarrier
    
    使用场景：多个线程彼此等待，当所有的线程都到达指定“地点”（指定代码位置），才开始继续执行。
    
    Semophore
    
    使用场景：需要控制访问某个资源或者进行某种操作的线程数量。当达到指定数量时，只能等待其他线程释放信号量。
~~~
